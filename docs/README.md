## 技术栈速览

```
前端: React 18 + TypeScript + Vite + TailwindCSS + Zustand + Socket.IO Client
后端: Node.js 20 + Express + Socket.IO + TypeScript + Prisma
数据: PostgreSQL 16 + Redis 7
结构: pnpm Monorepo (packages/shared + server + client)
部署: Docker + Docker Compose + Nginx
```

## MVP 范围

- 一副牌版（52张，4人）
- 游客登录 + 房间制（创建/加入/邀请链接）
- 完整游戏循环：发牌 → 出牌 → 回合结算 → 拖三/憋三 → 终局结算
- 13种牌型识别与比较
- 基础 UI（手机优先响应式）
