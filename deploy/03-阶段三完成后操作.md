# 阶段三：核心游戏逻辑（Phase 2）

## 本阶段将完成的内容

- **P2-1** 牌堆与发牌（Deck 类、Fisher-Yates 洗牌）
- **P2-2** 牌型检测（13 种牌型识别）
- **P2-3** 牌型比较（同类型比较 + 跨类型压制）
- **P2-4** 出牌验证（轮次、持牌、牌型、接风限制）
- **P2-5** 拖三与憋三规则
- **P2-6** 计分系统
- **P2-7** 游戏引擎（状态机）

## 验证标准

```bash
cd packages/server
npx vitest run
```

所有测试应通过，覆盖以下要点：

- 52 张牌无遗漏无重复
- 13 种牌型正确检测
- 压制关系表完整覆盖
- 拖三/憋三判定准确
- 完整模拟对局结算正确

> 本阶段是纯逻辑代码，不依赖数据库和 Redis，可以在没有基础设施的环境下开发和测试。
