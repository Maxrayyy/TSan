# 生产部署指南（Linux 云服务器）

## 1. 服务器初始化

```bash
# 更新系统
sudo apt update && sudo apt upgrade -y

# 安装 Node.js 20
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

# 安装 pnpm 和 PM2
npm install -g pnpm pm2

# 安装 Nginx
sudo apt install -y nginx

# 安装 PostgreSQL 16
sudo apt install -y postgresql postgresql-contrib

# 安装 Redis 7
sudo apt install -y redis-server
sudo systemctl enable redis-server
```

## 2. 数据库配置

```bash
sudo -u postgres psql <<EOF
CREATE USER tuosan WITH PASSWORD 'your_strong_password_here';
CREATE DATABASE tuosan OWNER tuosan;
EOF
```

## 3. 项目部署

```bash
# 克隆代码
cd /opt
git clone <your-repo-url> tuosan
cd tuosan

# 安装依赖
pnpm install --frozen-lockfile

# 配置环境变量
cp .env.example .env
nano .env
# 修改为生产配置：
# DATABASE_URL=postgresql://tuosan:your_strong_password_here@localhost:5432/tuosan
# REDIS_URL=redis://localhost:6379
# JWT_SECRET=<生成一个64位随机字符串>
# JWT_REFRESH_SECRET=<生成另一个64位随机字符串>
# PORT=3000
# NODE_ENV=production

# 构建
pnpm --filter @tuosan/shared build
cd packages/server && npx prisma db push && cd ../..
pnpm --filter @tuosan/server build
pnpm --filter @tuosan/client build
```

## 4. PM2 启动后端

```bash
pm2 start packages/server/dist/index.js --name tuosan-server
pm2 save
pm2 startup  # 按提示执行输出的命令，实现开机自启
```

## 5. Nginx 配置

```bash
sudo nano /etc/nginx/sites-available/tuosan
```

写入：

```nginx
server {
    listen 80;
    server_name your-domain.com;

    # 前端静态文件
    location / {
        root /opt/tuosan/packages/client/dist;
        try_files $uri $uri/ /index.html;
    }

    # API 代理
    location /api/ {
        proxy_pass http://127.0.0.1:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    # Socket.IO 代理
    location /socket.io/ {
        proxy_pass http://127.0.0.1:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

```bash
sudo ln -s /etc/nginx/sites-available/tuosan /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

## 6. HTTPS（可选但推荐）

```bash
sudo apt install -y certbot python3-certbot-nginx
sudo certbot --nginx -d your-domain.com
```

## 7. 常用运维命令

```bash
# 查看后端日志
pm2 logs tuosan-server

# 重启后端
pm2 restart tuosan-server

# 更新代码后重新部署
cd /opt/tuosan
git pull
pnpm install --frozen-lockfile
pnpm --filter @tuosan/shared build
pnpm --filter @tuosan/server build
pnpm --filter @tuosan/client build
pm2 restart tuosan-server
```
