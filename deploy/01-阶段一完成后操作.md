# 阶段一完成后操作（项目基础设施 Phase 0）

## 已完成的内容

- Monorepo 结构（pnpm workspace）
- Docker Compose 配置
- 后端 Express 骨架 + 日志 + 错误处理
- 前端 Vite + React + TailwindCSS 骨架
- 共享类型包 @tuosan/shared

## 你需要做的操作

### 1. 安装依赖

```bash
cd d:\code\TSan
pnpm install
```

### 2. 构建共享包

```bash
cd packages/shared
pnpm build
```

### 3. 启动基础设施（Docker 方案）

```bash
docker compose up -d
```

这会启动 PostgreSQL (5432) 和 Redis (6379)。

### 4. 验证

```bash
# 后端
cd packages/server
pnpm dev
# 浏览器访问 http://localhost:3000/health 应返回 {"status":"ok",...}

# 前端（另一个终端）
cd packages/client
pnpm dev
# 浏览器访问 http://localhost:5173 应看到首页
```
