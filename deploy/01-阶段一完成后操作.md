# 阶段一：项目基础设施（Phase 0）

## 本阶段完成的内容

- Monorepo 结构（pnpm workspace + 三个包）
- Docker Compose 配置（PostgreSQL 16 + Redis 7）
- 后端 Express 骨架（路由、日志、错误处理、健康检查）
- 前端 Vite + React + TailwindCSS 骨架
- 共享类型包 @tuosan/shared（类型定义 + 常量）
- 代码规范工具链（ESLint + Prettier + Husky + lint-staged）

## 验证标准

- `pnpm build:shared` 构建成功
- 后端 `pnpm dev:server` 启动后，`http://localhost:3000/health` 返回 `{"status":"ok",...}`
- 前端 `pnpm dev:client` 启动后，`http://localhost:5173` 显示首页
