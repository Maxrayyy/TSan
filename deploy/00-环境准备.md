# 环境准备

## 本地开发环境（Windows）

### 必需软件

1. **Node.js >= 20 LTS** — https://nodejs.org/
2. **pnpm >= 9** — `npm install -g pnpm`
3. **Docker Desktop for Windows** — https://www.docker.com/products/docker-desktop/
   - 安装后需启用 WSL2 后端
   - 如果不想装 Docker，也可以直接安装 PostgreSQL 和 Redis（见下方"无 Docker 方案"）

### 无 Docker 方案（Windows 本地直装）

**PostgreSQL 16：**

1. 下载安装 https://www.postgresql.org/download/windows/
2. 安装时设置 superuser 密码
3. 安装完成后用 pgAdmin 或 psql 创建数据库：
   ```sql
   CREATE USER tuosan WITH PASSWORD 'tuosan_dev';
   CREATE DATABASE tuosan OWNER tuosan;
   ```

**Redis 7：**

- Windows 原生不支持 Redis 7，有两种方案：
  1. **推荐：Memurai**（Redis 兼容）— https://www.memurai.com/ 免费社区版
  2. **WSL2 中安装**：`wsl` 进入 Linux 后 `sudo apt install redis-server`
  3. **云 Redis**：用阿里云/腾讯云的免费试用 Redis 实例

### 环境变量

复制 `.env.example` 为 `.env` 并填入实际值：

```bash
cp .env.example .env
```

`.env` 内容：

```
DATABASE_URL=postgresql://tuosan:tuosan_dev@localhost:5432/tuosan
REDIS_URL=redis://localhost:6379
JWT_SECRET=your-random-secret-at-least-32-chars
JWT_REFRESH_SECRET=another-random-secret-at-least-32-chars
PORT=3000
NODE_ENV=development
```

## 生产环境（Linux 云服务器）

### 推荐配置

- Ubuntu 22.04 / Debian 12
- 2核 4G 内存（最低 1核 2G）
- Node.js 20 LTS + pnpm
- PostgreSQL 16（可用云数据库 RDS）
- Redis 7（可用云 Redis）
- Nginx（反向代理 + 静态资源）
- PM2（Node.js 进程管理）

### 安装步骤概要

```bash
# Node.js
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs
npm install -g pnpm pm2

# PostgreSQL
sudo apt install -y postgresql postgresql-contrib
sudo -u postgres psql -c "CREATE USER tuosan WITH PASSWORD 'your_prod_password';"
sudo -u postgres psql -c "CREATE DATABASE tuosan OWNER tuosan;"

# Redis
sudo apt install -y redis-server
sudo systemctl enable redis-server

# Nginx
sudo apt install -y nginx
```
